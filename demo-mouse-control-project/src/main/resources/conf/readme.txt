使用说明：

软件提供5个点，启动软件时，鼠标会不断滑动这5个点。

软件提供两个鼠标点击点，启动软件时，鼠标会在time时间间隔内，点击所在的坐标。


shift+s启动

shift+p暂停

shift+l位置
shift+x退出
